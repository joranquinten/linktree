# JQ Linktree

A modern, responsive personal linktree for [Joran Quinten](https://joranquinten.nl), built with pure HTML, CSS, and JavaScript. This project showcases links to professional profiles, projects, and contact information, with a beautiful theme toggle and animated UI.

## Features

- **Responsive Design:** Works seamlessly on desktop and mobile.
- **Theme Toggle:** Switch between dark and light modes, with preference saved in localStorage.
- **Animated UI:** Smooth fade-in and float animations for cards and avatar.
- **Colorful Cards:** Each link card features a unique accent color and hover effect.
- **Contact Section:** Easy access to email and location info.
- **No Dependencies:** Pure HTML, CSS, and JavaScript—no frameworks required.

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/joranquinten/linktree.git
   cd linktree
   ```
2. **Open `index.html` in your browser.**

No build steps or dependencies required.

## Customization

- **Links:** Edit the `<div class="links-container">` in `index.html` to add, remove, or update your links.
- **Colors & Styles:** Modify `styles.css` for custom color schemes or layout tweaks.
- **Avatar:** Update the `.avatar` div for initials or replace with an image if desired.
